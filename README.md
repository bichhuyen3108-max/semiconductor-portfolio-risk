# 한국 반도체 포트폴리오 시장위험 정량 분석 연구

VaR 및 스트레스 테스트 기반 리스크 정량 분석 프로젝트

## 📌 Project Overview
본 프로젝트는 한국 반도체 산업 주요 기업(Samsung Electronics, SK Hynix, Samsung SDI)을 대상으로 포트폴리오 시장위험을 정량적으로 분석하는 것을 목표로 한다.

VaR 및 스트레스 테스트 기법을 활용하여 하방 리스크를 측정하고, 다양한 위험 측정 방법론을 비교 분석한다.

---

## 🎯 Research Objective
- 시장위험 정량적 측정
- VaR 및 CVaR(Expected Shortfall) 비교
- Monte Carlo Simulation 기반 리스크 분석
- Backtesting을 통한 모델 검증

---

## 🛠 Tech Stack
- Python
- Pandas
- Numpy
- Matplotlib / Seaborn

---

## 📊 Methodology
- Historical VaR
- Parametric VaR
- Monte Carlo Simulation
- CVaR
- Backtesting
